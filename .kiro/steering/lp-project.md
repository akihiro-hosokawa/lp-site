# 名古屋・大阪支社 採用LP プロジェクト方針

## 概要
当社の名古屋・大阪支社の求職者向けランディングページを制作中。
ITエンジニアの派遣・SESによる客先常駐型の支援事業を行う支社のページ。

## 採用方針
- 中途採用のみ（未経験・経験の浅い方は対象外）
- 入社時のスキルレベルは人によって異なる
- 「未経験歓迎」等の表現は使用しない

## キャリアパスの方針
- 年次ベース（1年目→3年目→5年目）のキャリアパスは使わない
- ロールベースで「目指せる方向性」を示す形にする
- 現在の4ロール：
  1. テックリード（技術で組織を牽引）
  2. プロジェクトリーダー / PM（マネジメント寄り）
  3. インフラ・クラウドスペシャリスト（基盤の専門家）
  4. ITコンサルタント / 上流特化（課題解決・提案）
- 「フルスタックエンジニア」は使わない

## 技術スタック
- HTML + CSS + JavaScript（フレームワークなし）
- Tailwind CSS（CDN利用）
- レスポンシブ対応

## ファイル構成
- `index.html` - メインHTML
- `style.css` - カスタムスタイル
- `script.js` - インタラクション（アニメーション、FAQ等）
